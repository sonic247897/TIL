# 0장. 정보기술의 원리와 철학

컴퓨터 및 정보기술 분야는 매우 빠른 변화를 겪는 학문이다. 하지만 `컴퓨터 분야의 기본 원리` 및 `정보기술 분야가 추구하는 핵심 철학`은 시대가 흘러도 변하지 않는다. 

> 1937년 수학자 앨런 튜링이 개발한 이론적 컴퓨터인 **튜링머신**이 현대 컴퓨터가 풀 수 있는 모든 문제를 풀 수 있다는 사실이 증명되었다.



# 1장. 컴퓨터 및 정보기술의 역사

> 운영체제를 설명하기에 앞서 컴퓨터와 정보기술 분야의 역사를 살펴본다.

## 1. 컴퓨터의 선사시대

### 1) 이론적인 컴퓨터의 역사

컴퓨터의 이론적인 기원은 `수학`과 `논리학`에서 그 기초를 찾아볼 수 있다.

- 컴퓨터의 이론적인 임무
  - 숫자의 표현
  - 숫자의 처리 방식

1930년대 컴퓨터에 대한 추상적인 모델

- 튜링머신(Turing Machine) : 컴퓨터에 대한 가자 단순한 이론적 모델
- 처치의 가정(Church's thesis) : 이론적 컴퓨터인 `튜링머신`이 현대 컴퓨터가 풀 수 있는 모든 문제를 풀 수 있다는 사실이 증명되었다. 처치의 가정은 튜링머신보다 더 처리력이 우수한 컴퓨터 구조는 있을 수 없다고 말하고 있다.

컴퓨터 형식체계의 한계

- 튜링머신의 정지 문제(halting problem) : 튜링 머신과 주어진 입력 데이터를 갖고 동작시킨 결과, 튜링 머신이 **처리를 마친 후 정지할 것인지, 아니면 정지하지 않고 무한 루프에 빠질 것인지**를 알 수 있는가 하는 문제

  => 컴퓨터를 이용해서는 그 답을 알 수 없음

  => 이러한 결과는 `튜링머신의 정지 문제` 외에도, **컴퓨터로 풀 수 없는 영역이 존재함을 의미함**



### 2) 기계식 컴퓨터의 역사

19세기에 개발된 해석기관(analytic engine)은 프로그램이 가능한 최초의 기계식 컴퓨터이며, 오늘날 사용 중인 모든 컴퓨터가 `해석기관의 기본 구조`를 따르고 있다. 수학자인 찰스 배비지(Charles Babbage)가 고안하였다.

- 현대의 모든 컴퓨터에서 발견되는 네 가지 기본 구성 요소 포함
  - 입력장치
  - 출력장치
  - 처리장치
  - 저장장치



### 3) 전자식 컴퓨터의 초기 역사

ENIAC은 `최초의 현대적 컴퓨터`로 인식되고 있으며, 컴퓨터 역사를 분류할 때 선사시대와 역사시대의 경계를 ENIAC의 탄생 시점으로 보는 경향이 많다.



## 2. 근대적 컴퓨터의 역사

매우 빠른 변화가 일어났으며 대표적으로 `하드웨어의 발전`을 토대로, 1세대부터 4세대까지 분류한다.

### 1) 1세대

> 1940년대 후반~1950년대 초반
>
> 진공관 기반 컴퓨터

ENIAC, UNIVAC 1

기계어 프로그래밍(숫자) - **동작 순서**를 나타내는 언어



### 2) 2세대

> 1950년대 후반~1960년대 초반
>
> 트랜지스터 기반 컴퓨터

어셈블리 언어 등장(기호) - 좀 더 사람에게 친숙하며 **동작 순서**를 나타내는 언어

↓

포트란Fortran 등장(고급 언어) - 사람이 표현하고자 하는 **문제** 자체에 더 가까운 언어 

리습Lisp - 기호 정보를 처리

코볼Cobol - 사무적인 정보 처리

`구조적 프로그래밍(structured programming)` 기법

- 인간이 이해하기 쉬운 `작은 단위`들로 나누어 **각 단위를 독립적으로 프로그래밍**하는 기법

**운영체제 개발**

- 초기에는 `컴퓨터 외부`에서 미리 예약해 한꺼번에 처리하는 **일괄처리 방식(batch processing)**을 사용했으나, 이러한 방법이 비효율적임을 인식하고 `컴퓨터 자체가 이를 자동적으로 처리해주는 방식`을 고안했다. 

  ​	**=> 운영체제**



### 3) 3세대

> 1960년대 중반~1970년대 초반
>
> 집적회로(IC) 기반 컴퓨터

**무어의 법칙(Moore's law)**

- 반도체 기업 인텔INTEL의 회장이었던 무어는 반도체 IC의 성능이 적어도 20년 동안은 `1.5년마다 두 배씩` 발전할 것이라고 예견함
- 2010년대까지 정확하게 맞아떨어졌다.

개인용 컴퓨터(PC)

C언어 등장

- UNIX 운영체제의 개발과 밀접하게 관련
- UNIX의 등장으로 인해 여러 가지 `플랫폼`을 지원해주는 환경이 마련됨



### 4) 4세대

> 1970년대 중반~
>
> VLSI(초고밀도 집적회로) 기반 마이크로컴퓨터 - 인텔이 개발한 **마이크로프로세서**를 사용함

PC의 폭발적 보급

- 애플(Apple), 코모도어(Commodore), 탠디(Tandy)

병렬 처리와 분산 처리

멀티미디어

`객체지향 언어(object-oriented language)` 등장

- 신뢰성과 재사용성이 높은 소프트웨어 개발

클라이언트-서버 구조의 데이터 처리 방식 발전

인터넷 확산



## 3. 현대의 컴퓨터

임의의 목적으로 사용될 수 있는`범용 컴퓨터`와, 특수한 목적을 위해 각종 장치의 제어용으로 내장되는 `임베디드 컴퓨터`가 있다. 이들은 기본적인 구조에는 큰 차이가 없지만 상이한 특징을 가진다.

### 1) 슈퍼컴퓨터와 메인프레임 컴퓨터

메인프레임 컴퓨터

- 은행이나 항공사 같은 대규모 사업장, 수학과 과학의 연구용

- 일반적으로 터미널(terminal)을 통해서 접속해 사용함
  - 터미널 : 입력을 위한 키보드 + 출력을 위한 모니터
  - 요즘은 PC를 통해 터미널 업무를 수행하기도 한다.
- `시분할(time sharing) 방식`을 사용
  - 시분할 방식 : 컴퓨터 처리 능력을 **짧은 시간 단위**로 구분하고 여러 사용자에게 조금씩 분할해 서비스하는 방식

슈퍼 컴퓨터

- 기상 예측, 통신망 설계, 석유 탐사 등 메인프레임 컴퓨터로도 시간이 오래 걸리는 복잡한 문제에 사용
- 최근에는 슈퍼 컴퓨터를 원격으로 사용하고 사용료를 지불하는 `클라우드 컴퓨팅(cloud computing)`이 각광받고 있다.
  - 아마존(Amazon), 구글(Google), 마이크로소프트(Microsoft)
  - 각 사용자 입장에서는 자신만의 독자적인 컴퓨터가 존재하는 것처럼 인식된다. 
    - `가상 머신(virtual machine)` : 사용자가 느끼는 독자적인 컴퓨터 
    - `가상화(virtualization) 기술` : 가상 머신을 지원해주는 기술



### 2) 개인용 컴퓨터

워크스테이션(workstation)

- 개인용 컴퓨터를 대형화한 컴퓨터
- `서버(server)` : 서버는 하드웨어를 지칭하는 용어라기보다, **네트워크로 연결된 다른 컴퓨터에 서비스를 제공하는 `소프트웨어`**를 지칭하는 용어
  - 서버 컴퓨터는 보통 워크스테이션 이상 급의 컴퓨터가 담당한다.

개인용 컴퓨터

- 성능이 향상되어 개인용 컴퓨터가 서버용으로 사용되기도 한다.



### 3) 휴대용 컴퓨터

노트북 컴퓨터

스마트 디바이스

- 스마트폰, 태블릿
- PC나 노트북과는 다른 종류의 `모바일 운영체제`가 설치되며, 이에 따라 사용되는 프로그램들도 `애플리케이션(application)` 또는 `앱(app)`이라고 불리는 독자적인 프로그램들로 구성된다.
- 다양한 센서 장치가 부착됨



### 4) 임베디드 컴퓨터

특수한 목적을 가지고 제작되는 컴퓨터

보통 기기 내부에 장착되는 형태가 된다.

임베디드 컴퓨터는 **프로그램이 칩 내부에 구워져 있어서** 일반적인 방법으로는 새로운 프로그램을 탑재할 수 없다.

(용도의 특수성 때문에 프로그램이 수정될 일이 거의 발생하지 않기 때문)
